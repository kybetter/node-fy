#### 介绍
一款命令行翻译工具，目前只接入了有道翻译。
没有使用其他第三方库，是为了使这个工具更小巧，依赖更少。
你只需要下载`index.js`文件就可以在命令行使用它了，无需再安装其他依赖项。

#### 使用方法

#### 1、安装
```
// 安装到全局环境
$ npm i -g @kybetter/node-fy
或
$ yarn add global @kybetter/node-fy

// 单个目录中使用
$ npm i @kybetter/node-fy
或
$ yarn add @kybetter/node-fy
```
